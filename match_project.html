Présentation

Projet qui consiste à prédire les scores des matchs de football en utilisant les données de la saison précédente

Objectif

Prédire les scores des matchs de football en utilisant les données de la saison précédente

Données

Les données utilisées pour ce projet sont des données de la saison 2019/2020

Modèle

Le modèle utilisé pour ce projet est un modèle de régression linéaire

Resultats

Le modèle a un score de 0.95
